# 🇷🇸 Учим сербский с Claude

Персональный тренажёр сербского языка с упором на **разговорную речь**.
Заниматься можно прямо с телефона — через приложение GitHub или чат с Claude.

## Как это работает

Claude — твой преподаватель. Ты открываешь чат и говоришь:

> **«Давай занятие»** — и начинается урок.

Claude сам:
- посмотрит твой прогресс в [`progress/progress.md`](progress/progress.md);
- проведёт занятие по плану из [`lessons/curriculum.md`](lessons/curriculum.md);
- даст разговорную практику, новые слова и грамматику;
- запишет результаты: обновит прогресс, словарь и журнал.

Тебе не нужно ничего настраивать — просто пиши с телефона.

## Полезные команды (просто пиши их в чат)

| Команда | Что делает |
|---|---|
| `Давай занятие` | Полноценный урок (~15–25 мин) |
| `Быстрая разминка` | 5 минут: повторить слова + пара фраз |
| `Поговорим по-сербски` | Свободный диалог на пройденные темы |
| `Проверь меня` | Тест по словам и грамматике (spaced repetition) |
| `Объясни <тема>` | Разбор грамматики или фразы |
| `Как сказать "<фраза>"` | Перевод + произношение + варианты |
| `Мой прогресс` | Краткий отчёт: уровень, серия, что дальше |

## Структура проекта

```
lessons/      План обучения и шаблон урока
vocabulary/   Словарь с интервальным повторением
grammar/      Накопленный справочник по грамматике
phrases/      Разговорные фразы по темам
progress/     Прогресс, серия занятий и журнал уроков
CLAUDE.md     Инструкция для Claude: как вести занятие
```

## Текущие настройки

- **Уровень:** с нуля (A0 → A1)
- **Родной язык:** русский (используем близость языков 🚀)
- **Письмо:** латиница сейчас, кириллица — постепенно
- **Норма:** белградский стандарт (екавица)
- **Фокус:** живая разговорная речь, а не зубрёжка

> Любую настройку можно поменять — просто скажи Claude.
