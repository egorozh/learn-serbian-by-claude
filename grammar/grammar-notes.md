# 📚 Грамматика (справочник)

Накапливается по мере занятий — только то, что реально нужно для разговора.
Без огромных таблиц: короткое правило + примеры.

## Алфавит и произношение

Сербский пишется и **латиницей**, и **кириллицей** — один к одному, буква в букву.
Как слышится, так и пишется (фонетический принцип).

Особые буквы (латиница → кириллица → звук):
- `č` → ч → твёрдое «ч» (как русское ч в «чай»)
- `ć` → ћ → мягкое «чь» (мягче, ближе к «ть»)
- `š` → ш → ш
- `ž` → ж → ж
- `dž` → џ → твёрдое «дж»
- `đ` → ђ → мягкое «джь»
- `j` → ј → й
- `lj` → љ → ль, `nj` → њ → нь

## Базовое (заполняется на занятиях)

_Здесь появятся: род существительных, глагол «быть» (jesam/sam), настоящее
время, vokativ для обращения, akuzativ и т.д. — по одному правилу за урок._

### Vokativ — обращение по имени (урок 1)

Когда обращаешься к человеку по имени, окончание меняется:
- Мужские имена на согласный: **Jegor → Jegore**, Ivan → Ivane
- Мужские имена на -o/-e: **Marko → Marko** (без изменений)

Пока просто запоминай на слух — правило придёт само.

### Глагол «быть» — короткие формы (понадобится сразу)

| | |
|---|---|
| ja **sam** | я есть |
| ti **si** | ты есть |
| on/ona **je** | он/она есть |
| mi **smo** | мы есть |
| vi **ste** | вы есть |
| oni **su** | они есть |

Пример: `Ja sam Marko.` — Я Марко. `Kako si?` — Как ты? (букв. «как ты есть»).

### Аккузатив после volim — «люблю что?» (урок 2)

После **volim** существительное стоит в аккузативе (винительный падеж).
Для большинства слов женского рода: окончание **-u**:
- muzik**a** → Volim muzik**u**
- košark**a** → Volim košark**u**

Глагольная конструкция «люблю делать»: **volim da + глагол**:
- Volim da igram košarku. — Я люблю играть в баскетбол.
- Volim da slušam muziku. — Я люблю слушать музыку.

### Предлог iz — «из» (урок 3)

**iz** + родительный падеж (генитив) страны/города:
- Ja sam iz Rusije. — Я из России. (Rusija → Rusije)
- Ja sam iz Srbije. — Я из Сербии. (Srbija → Srbije)
- Ja sam iz Moskve. — Я из Москвы. (Moskva → Moskve)

Закономерность: -a → -e (как в русском -а → -и, но проще)

### Счёт лет — imam + godina/godine/godinu (урок 2–3)

- imam 1 godin**u** (аккузатив ед.ч.)
- imam 2/3/4 godin**e** (генитив ед.ч.)
- imam 5+ godin**a** (генитив мн.ч.)
- Числа 21, 31, 41... → как 1: **godinu**
- Числа 22–24, 32–34... → как 2–4: **godine**

### Аккузатив при заказе — «jednu + существительное» (урок 4)

При заказе (molim, hoću) существительные женского рода стоят в аккузативе (-a → -u):
- kaf**a** → jednu kaf**u**, molim
- vod**a** → jednu vod**u**, molim
- pljeskavic**a** → jednu pljeskavic**u**

Прилагательное тоже меняется: velik**a** → velik**u** pljeskavicu.
Правило то же, что после volim — аккузатив везде одинаковый.

### Аккузатив при покупке — «daj mi / hoću + существительное» (урок 5)

То же правило работает в магазине — после daj mi, hoću, treba mi:
- jabuk**a** → Daj mi jabuk**u**
- šargarep**a** → Hoću jednu šargarep**u**
- vod**a** → Treba mi vod**a** (treba mi — особый оборот, часто без аккузатива)

Ученик интуитивно использовал аккузатив без объяснения правила — хороший знак!

### Вежливость: извини vs изvinite (урок 5)

- **Izvini** — к другу, на «ты» (ты-форма)
- **Izvinite** — к незнакомым, официантам, продавцам (вы-форма)

Аналогично русскому: «извини» (ты) / «извините» (вы).
