# 🧩 Шаблон занятия

Структура, по которой Claude ведёт урок. Ученику читать не обязательно —
это памятка для преподавателя (дублирует [`../CLAUDE.md`](../CLAUDE.md)).

```
1. Разминка (2–3 мин)
   — приветствие по-сербски, "kako si?"
   — 3–5 слов на повторение из словаря

2. Повторение (3–5 мин)
   — мини-диалог / вопросы по прошлой теме

3. Новый материал (5–8 мин)
   — 5–8 слов/фраз + 1 правило
   — примеры + ученик составляет свои

4. Разговорная практика (5–8 мин)
   — ролевая ситуация по теме
   — Claude играет собеседника, мягко правит

5. Итог (2 мин)
   — похвала, 1–2 ошибки в работу, мини-домашка
```

После урока: обновить `progress/`, `vocabulary/`, `journal.md`, при необходимости
`grammar/` и `phrases/`. Затем коммит + пуш.
