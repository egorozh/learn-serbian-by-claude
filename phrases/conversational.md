# 💬 Разговорные фразы по темам

Готовые фразы для живого общения. Claude пополняет после занятий.
Произношение — кириллицей в скобках.

## Приветствие и прощание

| Сербский | Произн. | Перевод |
|----------|---------|---------|
| Zdravo! / Ćao! | здра́во / ча́о | Привет! / Пока! |
| Dobro jutro | до́бро ю́тро | Доброе утро |
| Dobar dan | до́бар дан | Добрый день |
| Dobro veče | до́бро ве́че | Добрый вечер |
| Doviđenja | довиђе́ња | До свидания |
| Vidimo se! | ви́димо се | Увидимся! |

## Вежливость

| Сербский | Произн. | Перевод |
|----------|---------|---------|
| Hvala (lepo) | хва́ла (ле́по) | Спасибо (большое) |
| Molim | мо́лим | Пожалуйста / не за что |
| Izvini / Izvinite | изви́ни / извини́те | Извини / извините |
| Nema problema | не́ма пробле́ма | Нет проблем |
| Drago mi je | дра́го ми је | Приятно познакомиться |

## Не понимаю (спасательные фразы)

| Сербский | Произн. | Перевод |
|----------|---------|---------|
| Ne razumem | не ра́зумем | Я не понимаю |
| Ponovi, molim | по́нови мо́лим | Повтори, пожалуйста |
| Sporije, molim | спо́рије мо́лим | Помедленнее, пожалуйста |
| Kako se kaže...? | ка́ко се ка́же | Как сказать...? |
| Šta znači...? | шта зна́чи | Что значит...? |
| Govorim malo srpski | гово́рим ма́ло ср́пски | Я немного говорю по-сербски |

## О себе

| Сербский | Произн. | Перевод |
|----------|---------|---------|
| Radim kao... | ра́дим ка́о | Я работаю как / в качестве... |
| Imam N godina | и́мам ... го́дина | Мне N лет |
| Ja sam iz... | ја сам из | Я из... |
| Volim da... | во́лим да | Я люблю... (+ глагол) |
| Volim muziku / košarku | во́лим му́зику | Я люблю музыку / баскетбол |
| Šta voliš? | шта во́лиш | Что ты любишь? |
| Odakle si? | о́дакле си | Откуда ты? |
| Ja sam iz... | ја сам из | Я из... (+ страна в род. падеже) |
| Koliko imaš godina? | ко́лико и́маш го́дина | Сколько тебе лет? |
| Moj broj telefona je... | мoj брoj те́лефона је | Мой номер телефона... |
